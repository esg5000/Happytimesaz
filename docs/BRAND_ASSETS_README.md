# Brand Assets & Guidelines

## Folder Structure

### `/apps/web/public/assets/logos/`
Place your logo files here:
- `logo.svg` - Main logo (SVG preferred)
- `logo-horizontal.svg` - Horizontal variant
- `logo-icon.svg` - Icon-only variant
- `logo-white.svg` - White/light variant for dark backgrounds
- Any other logo variations

### `/apps/web/public/assets/images/`
Place brand images, patterns, backgrounds, etc. here.

### `/docs/`
Place brand documentation here:
- Brand book PDF
- Style guide
- Design tokens
- Any other brand guidelines
